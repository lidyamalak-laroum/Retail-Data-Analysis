# 👗 Women's Clothing Review Analysis

## Overview
End-to-end data analysis of 23K women's clothing 
e-commerce reviews including sentiment analysis, 
keyword extraction and interactive Power BI dashboard.

## Tools Used
Python (pandas, numpy, TextBlob) | Power BI | Git

## Key Insights
- 82% recommendation rate across all departments
- Trend department has lowest sentiment (-0.22)
- "Size" and "quality" are top concerns in negative reviews

## Project Structure
Data/ → Raw and processed datasets
Notebooks/ → Analysis pipeline
Dashboards/ → Power BI report
